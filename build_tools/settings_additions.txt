1. NEW DEFAULTS — add to `defaults` dict in load_settings():
-------------------------------------------------------------

defaults = {
    ...existing fields...,
    "use_pre_processor": True,          # enable pre-processor step
    "pre_processor_model": "",          # "" means: use model_a at runtime
}

NOTE on pre_processor_model default:
  Store "" in settings, resolve to settings["model_a"] at call time.
  This way, if the user changes model_a, the pre-processor follows automatically
  unless explicitly overridden.

  Resolution logic in collect_input():
      pp_model = settings.get("pre_processor_model") or settings.get("model_a", DEFAULT_MODEL_A)


2. TYPE VALIDATION — add to load_settings() coercion block:
------------------------------------------------------------

    # use_pre_processor
    defaults["use_pre_processor"] = bool(defaults.get("use_pre_processor", True))

    # pre_processor_model — must be valid model name or empty string
    pp_model = defaults.get("pre_processor_model", "")
