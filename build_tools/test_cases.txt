TEST 1 — Fragmented French / role + task (the original trigger case)
---------------------------------------------------------------------
Raw input:
  "Tu es l'Architecte-Ingénieur, rigoureux comme un allemand, avec un style de
   Général du bâtiment, et tu dois consolider ce pont fissuré en 3 étais
   structurels. Vas-y."

Expected restructured output (approximately):
  "Tu es un Architecte-Ingénieur senior, rigoureux et méthodique, avec l'autorité
   d'un Général du bâtiment. Ton rôle est d'analyser un pont fissuré et de
   proposer un plan de consolidation structurelle en exactement 3 étais (étayages)
   porteurs. Pour chaque étai : précise l'emplacement, le type de support, les
   matériaux, la charge estimée et la justification technique. Ton output est un
   rapport d'ingénierie clair, structuré, directement actionnable."

Verify:
  ✓ Role preserved (Architecte-Ingénieur)
  ✓ Constraint preserved (3 étais)
  ✓ Language preserved (French)
  ✓ Tone preserved (authoritative, rigorous)
  ✓ Output format inferred and stated (rapport structuré)
  ✓ Missing details filled (emplacement, matériaux, charge, justification)


TEST 2 — Short English fragment
--------------------------------
Raw input:
  "explain quantum computing"

Expected restructured output (approximately):
  "Explain quantum computing clearly and concisely. Cover: what it is, how it
   differs from classical computing (qubits vs bits, superposition, entanglement),
   current real-world applications, and its limitations. Target audience:
   technically literate non-specialist. Use concrete analogies where helpful."

Verify:
  ✓ Language preserved (English)
  ✓ Input expanded (was < 5 words)
  ✓ Structure imposed (what/how/applications/limits)
  ✓ Audience inferred
  ✓ No answer given — only restructured prompt


TEST 3 — Mixed language, typos
--------------------------------
Raw input:
  "/expert /raisonnement Build une REST API en python avec authentification
   and explaique moi how to deploy it sur un VPS"

Expected restructured output (approximately):
  "/expert /raisonnement
   You are a senior Python backend engineer. Design a production-ready REST API
   in Python (FastAPI or Flask, your choice — justify it) with JWT authentication.
   Cover: project structure, authentication flow (registration, login, token
   refresh), protected endpoint example, and step-by-step deployment on a Linux
   VPS (systemd service + nginx reverse proxy + HTTPS). Output: annotated code
   snippets + deployment checklist."

Verify:
  ✓ /slash metacommands preserved (expert, raisonnement)
  ✓ Language unified (English dominant — user mixed FR/EN, both valid)
  ✓ Typos corrected (explaique → explain)
  ✓ Technical gaps filled (auth type, deployment stack)
  ✓ Output format stated (code + checklist)
