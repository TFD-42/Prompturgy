def pre_process_input(
    raw_input: str,
    model: str,
    ollama_url: str,
    timeout: int = 30,
    stream_callback=None,
) -> str:
    """
    Restructure raw user input into a clean, well-formed prompt using Ollama.
    Returns the restructured prompt, or raw_input unchanged on any failure.
    Never raises — all errors are silent fallbacks.
    """

IMPLEMENTATION SPEC:
--------------------

1. GUARD — skip if input is already clean/short enough
   if not raw_input or not raw_input.strip():
       return raw_input

   # If input is already very short (< 5 words) and has no obvious issues,
   # skip to save latency — caller decides, not this function.
   # The caller (collect_input) makes this decision via settings.

2. BUILD PROMPT
   from the PRE_PROCESSOR_PROMPT template (pre_processor_prompt_template.txt):
   prompt = PRE_PROCESSOR_PROMPT.replace("{RAW_INPUT}", raw_input.strip())

3. BUILD OLLAMA PAYLOAD
   payload = {
       "model": sanitize_input(model, "model"),
       "prompt": prompt,
       "stream": stream_callback is not None,
       "options": {
           "temperature": 0.1,
           "num_predict": 600,
           "stop": ["---", "<<<", "INPUT>>>"],
       },
   }

4. CALL OLLAMA (reuse existing generate_stream or inline a direct requests.post)
   Use the EXISTING stream/non-stream pattern already in the codebase:

   NON-STREAM PATH:
       resp = requests.post(
           ollama_url,
           json=payload,
           timeout=timeout,
           headers={"User-Agent": HTTP_USER_AGENT},
       )
       result = ""
       for line in resp.iter_lines():
           if not line:
               continue
           chunk = json.loads(line)
           result += chunk.get("response", "")
           if chunk.get("done"):
               break
       return sanitize_input(result.strip(), "text") or raw_input

   STREAM PATH (when stream_callback is not None):
       Same loop, but call stream_callback(chunk["response"]) each iteration.
       Accumulate into result string.
       Return sanitize_input(result.strip(), "text") or raw_input

5. FALLBACK — wrap everything in try/except
   try:
       ... (steps 3–4)
   except (requests.Timeout, requests.ConnectionError, json.JSONDecodeError,
           KeyError, ValueError, Exception):
       return raw_input   # transparent fallback — user never sees this error

-----------------------------
